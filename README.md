# mutatedSequence
get mutated protein sequence from all mutation(snp and indels)
## install
requirements
```
pandas
vcf
loguru
tqdm
biopython
bcftools
seqkit
csvkit
````

```sh
sh ./download.sh 109 # download seqnence GCRh38 109 release from ensembl
```
